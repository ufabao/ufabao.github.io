I am a machine learning researcher and mathematician. I received my Ph.D in Mathematics from Northwestern University in 2023, advised by Ben Antieau.

My mathematical research was focused on p-adic Hodge theory and homotopy theory. My thesis is titled 'Topological Restriction Homology is Locally Even in Characteristic p' and is available here. The research portion of my thesis is growing into a larger project, joint with Noah Riggenbach, that will appear soon.

Nowadays I am more focused on machine learning. Recently I built a transformer model by hand, trained on the LaTeX source code of the stacks project, which can generate LaTeX code that actually compiles into realistic looking algebraic geometry. Here is a pdf example of some of the math it has generated Fake Algebraic Geometry, and if you're interested in the code for the model here is the Github repository Algebraic Geometry Transformer.

To see more of my projects you can go to my Github.
